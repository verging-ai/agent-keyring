# Contributing to AgentKeyring

Thanks for your interest in contributing! AgentKeyring is in early stages, and all kinds of help are welcome.

## How to contribute

### Share feedback

The most valuable contributions right now are:

- Reporting setup pain points you've experienced with local AI tools
- Suggesting which tool integrations (connectors) would be most useful
- Filing bug reports or UX issues
- Helping validate and refine the MVP scope

Use the [Issue templates](https://github.com/YOUR_ORG/AgentKeyring/issues/new/choose) to get started.

### Code contributions

Once the codebase is more established, we'll welcome pull requests. For now:

1. Fork the repo
2. Create a feature branch (`git checkout -b my-change`)
3. Commit your changes with clear messages
4. Push and open a Pull Request

Please keep PRs focused and small when possible.

## Guidelines

- Be respectful and constructive
- Search existing issues before opening a new one
- Provide as much context as possible in bug reports
